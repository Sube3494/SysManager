# 📁 插件文件结构

```
SysManager/
├── main.py                 # 核心功能代码
├── metadata.yaml          # 插件元数据
├── requirements.txt       # Python依赖
├── _conf_schema.json      # 配置界面定义
├── README.md              # 完整文档
├── QUICKSTART.md          # 快速开始指南
├── 使用示例.md             # 详细使用示例
├── 配置示例.md             # 配置案例说明
└── 文件说明.md             # 本文件
```

## 📄 文件说明

### main.py
- **作用**：插件核心逻辑
- **功能**：
  - 命令执行和安全控制
  - 群组权限检查
  - 目录管理和会话保持
  - 输出优化和格式化
  - 交互式命令拦截

### metadata.yaml
- **作用**：插件识别信息
- **内容**：名称、版本、作者、描述

### requirements.txt
- **作用**：Python依赖声明
- **内容**：psutil（用于系统信息）

### _conf_schema.json
- **作用**：Web配置界面定义
- **配置项**：
  - enabled_groups - 群组白名单
  - command_timeout - 超时时间
  - max_output_length - 输出长度
  - log_operations - 日志开关

### README.md
- **作用**：完整使用文档
- **内容**：
  - 功能介绍
  - 安装方法
  - 配置说明
  - 使用示例
  - 安全提示
  - 故障排除

### QUICKSTART.md
- **作用**：新手快速入门
- **内容**：最简安装和使用步骤

### 使用示例.md
- **作用**：实际使用案例
- **内容**：各种场景的命令示例

### 配置示例.md
- **作用**：配置参考模板
- **内容**：不同场景的配置建议

## 🔧 配置文件位置

用户配置存储在：
```
data/config/SysManager_config.json
```

## 📝 日志位置

操作日志输出到AstrBot主控制台

## 🎯 快速定位

- **想了解功能** → README.md
- **想快速开始** → QUICKSTART.md
- **想看代码** → main.py
- **想改配置** → Web面板或 _conf_schema.json
- **想看示例** → 使用示例.md
